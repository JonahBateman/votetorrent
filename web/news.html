VoteTorrent News and Updates

Stay informed about the latest developments in the VoteTorrent project.

Founder Appears on The Crypto Podcast

March 6, 2025

GotChoices' founder is featured on Roy Coughlan's Crypto Podcast. This discussion includes the future of secure voting systems and how distributed ledger technology can revolutionize how we hold elections.

Watch on YouTube or listen on Spotify (starts at 50:03), or visit the Crypto Podcast Home Page

Authority Mobile App Development Underway

February 28, 2025

Development has begun on the Authority mobile application, a key component of the VoteTorrent ecosystem that will enable election administrators to securely manage voting events.

Distributed Database Passing Basic Tests

February 14, 2025

We've reached a significant milestone with the distributed database system passing basic testing scenarios. This component is crucial for ensuring vote data integrity and security across the peer-to-peer network and will also assist in future progress on the related MyCHIPs project.

Core Collection Implementation Complete

November 30, 2024

The initial implementation of the collection system is now complete. This system enables efficient storage and retrieval of ballots in a secure, verifiable manner across the peer-to-peer network.

VoteTorrent Project Launched

May 13, 2024

The VoteTorrent project is officially launched with its first code commit! This marks the beginning of our journey to create a secure, transparent, and community-driven voting system using peer-to-peer technology.
